## BLUE NOTE
재즈를 좋아하는 사람들을 위한 정보 공유 웹 애플리케이션

## 기술 스택
- nextJs
- tanstack-query,
- react-hook-form
- scss

## 배포 URL
https://blue-note-silk.vercel.app/

## 설치
```
npm install
```

## 개발환경 실행
```
npm run dev
```

## 주요 기능
1. 운영자 권한을 가진 유저의 공지사항, 뉴스 게시글 생성
2. 사용자들의 자유로운 게시글, 댓글의 작성, 수정, 삭제 등 (CRUD)
3. 회원가입, 로그인 기능
4. 모바일 환경과 데스크탑 환경을 모두 고려한 반응형 UI
